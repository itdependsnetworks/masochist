NAME
     wincent-icon-util -- apply a custom icon to a folder

SYNOPSIS
     wincent-icon-util -icon icns-file -target target-folder

DESCRIPTION
     wincent-icon-util applies the custom icon in icns-file to the folder
     target-folder.

OPTIONS
     -icon    Read custom icon from icns-file.

     -target  Apply custom icon to target-folder.

ERRORS
     On encountering an error wincent-icon-util will print a description to
     standard error and exit with an exit code of 1.

     GetFileInfo(1), SetFile(1), Rez(1)
