Downloading Hextrapolate

The latest version of Hextrapolate is always available from the Wincent website.

If you are running an older version of Mac OS X you may not be able to use the latest version of Hextrapolate. The downloads page indicates the minimum version requirement, and provides download links for previous versions of Hextrapolate that will run on older versions of Mac OS X.
