Reporting bugs

Before submitting a bug report please try the troubleshooting tips.

Submitting a bug report via the web is strongly recommended because you can use the web interface to perform a search before submitting (it may be that someone else has already reported the issue).

When submitting a bug report, please follow the guidelines for how to file a good bug report; submitting a detailed, high-quality bug report will help me to fix the issue.
