# `@masochist/benchmark`

## Usage

```
bun benchmark                  # Prints usage information.

bun benchmark:dynamic:lexer    # Runs the dynamic (legacy) lexer benchmarks.
bun benchmark:reference:lexer  # Runs the reference lexer benchmarks.
bun benchmark:static:lexer     # Runs the static lexer benchmarks.

bun benchmark:dynamic:parser   # Runs the dynamic (legacy) parser benchmarks.
bun benchmark:reference:parser # Runs the reference parser benchmarks.
bun benchmark:static:parser    # Runs the static parser benchmarks.
bun benchmark:table:parser     # Runs the table-based parser benchmarks.
```
